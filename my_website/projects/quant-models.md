# Quant Models Lab — 量化策略模型实验室

**Quant Models Lab** 是一个股票量化交易策略回测框架，包含因子分析、风险平价、蒙特卡洛模拟等核心模块。

## 核心特性

### 📊 因子分析
支持多因子模型构建与回测，包含动量因子、价值因子、质量因子等常见因子库。

### ⚖️ 风险平价
实现风险平价（Risk Parity）资产配置模型，自动计算最优权重分配。

### 🎲 蒙特卡洛模拟
基于 Monte Carlo 方法进行投资组合收益分布模拟，评估 VaR 与 CVaR 风险指标。

## 技术架构

- **语言**: Python
- **数据处理**: Pandas + NumPy
- **可视化**: Matplotlib
- **开发环境**: Jupyter Notebook
