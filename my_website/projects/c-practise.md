# C Code Workshop — C语言编程练习合集

**C Code Workshop** 是日常 C 语言编程练习仓库，包含算法实现、数据结构、系统编程等各类练习代码。

## 核心练习

### 🔢 算法实现
- 排序算法（快排、归并、堆排等）
- 搜索算法（二分查找、DFS、BFS）
- 数学问题（素数判断、矩阵转置等）

### 🏗️ 数据结构
- 链表、栈、队列
- 二叉树与遍历
- 哈希表

### ⚙️ 系统编程
- 文件 I/O 操作
- 内存管理
- 多线程基础

## 技术栈

- **语言**: C
- **编译**: GCC
- **构建**: Makefile
