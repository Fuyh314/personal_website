# Light & Shadow — 摄影作品集网站

**Light & Shadow** 是一个响应式摄影作品展示平台，具备瀑布流布局、图片懒加载、全屏画廊浏览功能。

## 核心特性

### 📱 响应式瀑布流
基于 CSS Columns 实现的瀑布流布局，在不同屏幕尺寸下自动调整列数，保证最佳的视觉展示效果。

### ⚡ 图片懒加载
使用 Intersection Observer API 实现图片懒加载，优化首屏加载速度和带宽消耗。

### 🖼️ 全屏画廊
点击任意图片即可进入全屏画廊模式，支持键盘导航、触控滑动、缩放查看等交互。

## 技术架构

- **前端**: HTML + CSS + JavaScript (Vanilla)
- **构建工具**: Vite
- **部署**: Vercel
